Objetivos

Diseño Web

Se implementará un desarrollo de servicios web de Quantum para el consumo de información de un Bot donde obtendremos las ventas en liquidación por parte de usuarios cliente.

Bases de Datos

A partir de la información obtenida de procederá con la creación de bases de datos relacionales que nos permitan construir el flujo operativo para el seguimiento de transacciones de cada usuario.

Seguridad

La información contenida en el sistema web principalmente información sensible como passwords y códigos de recompensas serán cifrados por un mecanismo de seguridad de desarrolo propio.
